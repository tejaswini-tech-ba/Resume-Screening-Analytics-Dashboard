# Resume-Screening-Analytics-Dashboard
Project Overview

This project analyzes recruitment data to help recruiters identify the most suitable candidates using Power BI and SQL.
The dashboard enables HR professionals to:
- Analyze applicant demographics
- Identify Best Match candidates
- Explore job-role-wise applications
- Filter applicants interactively
- Review candidate resumes

Business Problem
- Recruiters receive thousands of resumes for multiple job roles. Manually reviewing every applicant is time-consuming. This dashboard provides a data-driven approach to identify suitable candidates quickly.

Dataset
- 10,000 Applicants
- Job Roles
- Resume Information
- Gender
- Ethnicity
- Best Match 

Tools Used
- SQL
- Power BI
- Excel
